# vision-image-rotation

## image 영상처리 프로젝트
- 대학교 동아리 프로젝트 (https://blog.naver.com/korin57/220765312212)
- 기울어진 문서를 보정하기 위한 작업
1. 솔트 페퍼 잡음 제거를 위한 메디안 필터를 적용
2. 캐니 엣지 검출 적용
3. 직선을 활용 기울어진 각도를 계산 (왼쪽으로 기울시 -값, 오른쪽일시 +값 적용)
4. 각도를 통한 기존 이미지에 회전적용
